# PoetryQuiz

Quiz game on poetry.

Hi! =)

To start this program you will need **Python** version 3.0 and higher.

## Starting on Windows
Write in Command Line: 
```
python main.py
```
## Starting on Mac OS
Write in Terminal: 
```
python3 main.py
```
## Adding new poetry
Add poetry in _poetry.py_ like this:
```
"name_of_your_poetry": [
        "First line of poetry,",
        "Second line of poetry.",
    ]
```
